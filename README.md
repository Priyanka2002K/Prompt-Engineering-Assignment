# Prompt Engineering UI Assignment

This repository contains a complete **Prompt-Engineered Frontend UI Project** created by generating modern, responsive UI components using structured meta-prompts. The goal of this assignment is to demonstrate how strong prompts can be used to produce high-quality, reusable frontend sections.

---

##  Project Overview

This project showcases the following UI sections generated with prompt engineering:

* **Responsive Navigation Bar** (with hamburger menu)
* **Hero Section** (headline, subtext, actions, illustration)
* **Services / Features Section** (6 feature cards)
* **Contact Form Section** (fully responsive with form fields)
* **Footer Section** (4-column layout + social links)

The entire UI is built using **HTML + Tailwind CSS** and follows modern design principles: responsiveness, clean typography, spacing consistency, and accessibility.

---

##  Prompt Engineering Breakdown

The project includes a `meta prompt` that defines the global rules for generation. It also includes individual prompts for each UI section:

* Meta Prompt
* Navigation Bar Prompt
* Hero Section Prompt
* Services Section Prompt
* Contact Section Prompt
* Footer Prompt
* Combination Prompt (to generate a full index.html file)

These prompts are included at the top of the HTML file as documentation.
